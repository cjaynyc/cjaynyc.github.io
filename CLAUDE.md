# Project Instructions

## Skills System
This repo contains a shared skills library at `skills-audit-app/claude-skills/`.
- Global skills: `skills-audit-app/claude-skills/global/`
- Project skills: `skills-audit-app/claude-skills/projects/<name>/`
- Each skill is a folder with a `skill.md` file

Use `/audit` to scan for overlaps. Use `/fix-audit` to resolve issues. Use `/new-skill` to add skills.
