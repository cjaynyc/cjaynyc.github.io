# Longevity Stack — iOS (SwiftUI + HealthKit)

A native iOS port of the [web Longevity Stack](../longevity/), adding the one
thing a web app can't do: **Apple Health integration**.

## What HealthKit adds

| Capability | How it's used |
|---|---|
| **Read steps** | "Motion (Walk with Earl)" auto-completes at **6,000 steps** (configurable in `HealthManager.motionStepGoal`). |
| **Read active energy** | Today's kcal shown in the footer. |
| **Show steps** | A live "Steps Today" stat tile. |
| **Write Mindful Minutes** | Checking **Kumbhaka Breathwork** logs a 5-minute mindfulness session to Apple Health. |

Everything else (fasting ring, pillar checklist, streak, best-fast) mirrors the
web app, with state persisted locally via `UserDefaults`.

## Requirements

- A **Mac with Xcode 15+**
- An **iPhone** (HealthKit doesn't return data in the Simulator — auto-complete
  needs a real device with Health data)
- An Apple ID. A free one lets you side-load to your own iPhone for 7 days;
  a paid **Apple Developer account ($99/yr)** is needed for permanent installs
  / TestFlight / the App Store.

## Build it

### Option A — XcodeGen (recommended, generates the project for you)

```bash
brew install xcodegen      # once
cd ios
xcodegen generate          # creates LongevityStack.xcodeproj + Info.plist + entitlements
open LongevityStack.xcodeproj
```

In Xcode: select the **LongevityStack** target → **Signing & Capabilities** →
choose your Team (Xcode adds a free provisioning profile). Plug in your iPhone,
pick it as the run destination, and press ▶.

### Option B — Manual Xcode project (no XcodeGen)

1. **File ▸ New ▸ Project ▸ iOS ▸ App**, name it `LongevityStack`,
   interface **SwiftUI**, language **Swift**.
2. Delete the auto-generated `ContentView.swift` / `App.swift`, then drag every
   file from `Sources/` into the project (check *Copy items if needed*).
3. **Signing & Capabilities** → **+ Capability** → add **HealthKit**.
4. In the target's **Info** tab add these two keys (Xcode shows friendly names):
   - `Privacy - Health Share Usage Description` → *"Longevity Stack reads your
     step count and active energy to auto-complete your daily Motion pillar."*
   - `Privacy - Health Update Usage Description` → *"Longevity Stack logs your
     Kumbhaka breathwork as Mindful Minutes in Apple Health."*
5. Run on your iPhone. On first launch, approve the Health permission sheet.

> The `Sources/Info.plist` and `Sources/LongevityStack.entitlements` files are
> generated by XcodeGen (Option A). With Option B, Xcode manages those for you —
> you don't need to create them by hand.

## Notes & next steps

- HealthKit returns no data in the iOS Simulator — test on a device, or add
  sample steps in the Health app first.
- Possible follow-ons: log fasting windows, a 30-day history view, daily
  reminder notifications, a Lock Screen widget, or an Apple Watch companion.
