# Source

Vendored from [anthropics/launch-your-agent](https://github.com/anthropics/launch-your-agent) (Apache 2.0), commit `main` as of 2026-07-02.

This is a reference implementation skill for building [Claude Managed Agents](https://platform.claude.com/docs/en/managed-agents/overview). Not maintained upstream and not accepting contributions there — see `LICENSE` for terms.

To use: run Claude Code in this repo and type `/launch-your-agent`. Needs your own `ANTHROPIC_API_KEY` (never committed — goes in a local `my-agent/.env` the skill creates for you).
