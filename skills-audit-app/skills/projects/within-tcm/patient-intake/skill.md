# patient-intake

## Description
Send a new patient welcome email with intake forms and first-visit instructions for WITHiN TCM.

## Trigger
TRIGGER when: user asks to send intake forms or onboard a new patient

## References
- Email template in index.html (intake tab)
- Intake PDF forms (attached separately)

## Steps
1. Get the new patient name and email
2. Get the first appointment date/time
3. Select the office location
4. Generate the welcome email with intake instructions
5. Open in Apple Mail via mailto link

## Notes
Patient should receive this at least 48 hours before first visit.
