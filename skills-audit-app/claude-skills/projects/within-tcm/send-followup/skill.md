# send-followup

## Description
Send a followup email to a patient reminding them about their upcoming appointment at WITHiN TCM.

## Trigger
TRIGGER when: user asks to follow up with a patient about their appointment

## References

2. Get the appointment date/time
3. Generate the followup email
4. Send via Apple Mail
